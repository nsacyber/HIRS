# LDevID Certificate

An **[LDevID (Local Device Identity) Certificate :fontawesome-solid-external-link:](https://1.ieee802.org/security/802-1ar/){:target="_blank"}** is a digital identity created for a specific device after it has been successfully validated. 

<img src= "../../../images/bg-cert-ldevid.png" alt="LDevID Pic" style="float: right; width: 35%; margin-left: 20px; border: 1px solid grey;">

An LDevID can be thought of as a secure device identity issued only after the system successfully
passes a rigorous health and integrity check—similar to how a driver's license is issued only
after an individual demonstrates competence by passing a driving course.

The LDevID can be used for specific non-attestation applications such as joining an enterprise 
network. Once issued, it provides the device with a trusted, long-term identity, allowing other
systems on the network to verify that the device is genuine and operating in a verified state.
