# ACA Portal: Issued Certificates Page

The **Issued Certificates** page provides access to the Attestation Certificates (AC)
issued by the ACA. Note that there can be multiple ACs if the TPM 
provisioning process is run multiple times. 

<img src= "../../images/portal-issued.png" alt="Portal Platform Cert page" style="border: 2px solid grey;">

The user can view the details in the Issued Certificate by clicking the
<img src="../../images/portal-clipboard.png" alt="clipboard" width="20" height="20" style="vertical-align:middle;">
icon under the `Options` tab:
