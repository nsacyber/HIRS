# ACA Portal: Devices Page

The **Devices** page is similar to the Validation Reports page but only shows one row per device, thus 
allowing for easier access to a particular device’s status. As with the Validation Reports page, 
the credentials associated with the device are dictated by the ACA policy during the latest 
validation report.

<img src= "../../images/portal-devices.png" alt="Portal Platform Cert page" style="border: 2px solid grey;">

The green check indicates that the device passed validation. If a device does not pass, 
a red <span style="color: red;">**!**</span> will be shown.
