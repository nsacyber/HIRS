# ACA Portal: Help Page

The **Help** page displays the log that captures the validation processes and any errors that occur 
during the course of validating a device.

<img src= "../../images/portal-help.png" alt="Portal Help page" style="border: 2px solid grey;">

The user can use this page to download the log, and can also use this page to select the type of 
log level they would like to capture.
