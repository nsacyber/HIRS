# ACA Web Portal

The HIRS ACA features a Web Portal that contains support for managing trust chains, setting
validation policy, and viewing validation reports. After 
[installation of the ACA](../install/aca/aca-install.md), the
ACA portal can be accessed via a URL in a browser:

```shell
https://hostname:8443/
```

Where “hostname” is to be substituted with the name or IP address of the server that the ACA 
is installed on.

If you are new to HIRS and want a step-by-step tutorial, it is best to start with
the [Getting Started](../started/index.md) page.
